# VunaAI 🌿 - L'IA au Service de l'Agriculture Panafricaine

**VunaAI** ("Vuna" signifie "Récolte" en Swahili) est une Progressive Web App (PWA) de pointe conçue pour autonomiser les agriculteurs africains. En utilisant l'IA multimodale de Google (Gemini), VunaAI transforme n'importe quel smartphone en un expert agronome capable de diagnostiquer les maladies des cultures et de fournir des conseils stratégiques en temps réel.

## 🚀 Fonctionnalités Clés

- **Diagnostic par IA** : Scannez vos cultures pour identifier instantanément les maladies et carences.
- **Protocoles de Traitement Hybrides** : Recommandations basées sur l'agroécologie locale et les solutions conventionnelles.
- **Météo Agricole Locale** : Prévisions basées sur la géolocalisation pour optimiser les semis et l'irrigation.
- **Calendrier de Semis Panafricain** : Guide complet pour 12 cultures majeures adaptées aux différentes régions du continent.
- **Multilingue & Inclusif** : Disponible en Français, Anglais, Swahili, Wolof, Bambara, Hausa et Yoruba.
- **Suivi & Historique** : Gardez une trace de vos analyses passées pour surveiller l'évolution de votre exploitation.

## 🛠️ Stack Technique

- **Frontend** : React 19, Vite, Tailwind CSS.
- **Animations** : Motion (framer-motion).
- **Routage** : React Router 7.
- **IA** : Google Gemini Pro Vision via `@google/genai`.
- **Météo** : Open-Meteo API.
- **Icons** : Lucide React.

## 📦 Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-repo/vuna-ai.git
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Configurez les variables d'environnement :
   Créez un fichier `.env` à la racine et ajoutez votre clé API Gemini :
   ```env
   GEMINI_API_KEY=votre_cle_ici
   ```

4. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

## 🌍 Impact

VunaAI vise à réduire les pertes post-récolte de 30% en Afrique en fournissant un accès démocratisé à l'information agricole, favorisant ainsi la sécurité alimentaire et la résilience climatique.
