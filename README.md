# VunaAI 🌿 - L'IA au Service de l'Agriculture Panafricaine

**VunaAI** ("Vuna" signifie "Récolte" en Swahili) est une Progressive Web App (PWA) de pointe conçue pour autonomiser les agriculteurs africains. En utilisant l'IA multimodale de Google (**Gemini Pro Vision**), VunaAI transforme n'importe quel smartphone en un expert agronome capable de diagnostiquer les maladies des cultures et de fournir des conseils stratégiques en temps réel.

## 🚀 Fonctionnalités Clés

- **Diagnostic par IA (Computer Vision)** : Scannez vos cultures (maïs, manioc, cacao, etc.) pour identifier instantanément les maladies et carences via une analyse visuelle de pointe.
- **Protocoles de Traitement Hybrides** : Recommandations double Action : 
    - 🌿 *Agroécologie* : Solutions naturelles basées sur les ressources locales.
    - ⚗️ *Conventionnel* : Usage raisonné et sécurisé d'intrants.
- **Météo Agricole Locale** : Prévisions ultra-locales (Température, Humidité, Précipitations) pour une planification précise des travaux champêtres.
- **Calendrier de Semis Intelligent** : Guide pour 12 cultures majeures couvrant toutes les zones agro-climatiques de l'Afrique (Sahel, Équatoriale, Hautes Terres).
- **Inclusion Linguistique** : Interface disponible en Français, Anglais, Swahili, Wolof, Bambara, Hausa et Yoruba pour briser la barrière de la langue.
- **Système de Navigation Fluide** : Routage complet pour une expérience de navigation structurée et intuitive.
- **Gestion d'Historique** : Archive locale sécurisée pour suivre l'évolution de la santé de vos parcelles.

## 🛠️ Stack Technique

- **Core** : React 19 & Vite (Performance & Rapidité).
- **IA** : Modèle multimodal Gemini Pro de Google via le SDK `@google/genai`.
- **Routage** : React Router DOM v7 (Navigation SPA fluide).
- **Animations** : Motion/React (Transitions fluides et feedback visuel).
- **Design** : Tailwind CSS (UI Moderne, Responsive & Mobile-First).
- **Data Fetching** : Open-Meteo API & Geolocation API.

## 📦 Installation & Configuration

1. **Cloner le projet** :
   ```bash
   git clone https://github.com/glosings0n/VunaAI.git
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Variables d'environnement** :
   Créez un fichier `.env` à la racine :
   ```env
   GEMINI_API_KEY=votre_cle_api_gemini
   ```

4. **Lancer le développement** :
   ```bash
   npm run dev
   ```

## 🌍 Impact Social
VunaAI combat l'insécurité alimentaire en Afrique en mettant une expertise agronomique de niveau universitaire entre les mains des petits exploitants. La solution favorise la réduction des pesticides chimiques et l'augmentation des rendements par la précision numérique.
